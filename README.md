# 🏦 Console-Based Banking Application (Java)

## 📌 Project Overview
This is a console-based banking system developed in Java.  
It allows users to create accounts, login securely using a PIN, and perform banking operations such as deposits, withdrawals, fund transfers, and transaction history tracking.  
The project demonstrates core Java concepts like Object-Oriented Programming (OOP), encapsulation, collections, and menu-driven programming.

---

## 🎯 Features

- Create New Account with initial deposit  
- Secure Login using PIN  
- Deposit Money  
- Withdraw Money  
- Transfer Funds Between Accounts  
- Check Account Balance  
- View Transaction History  
- Logout  

---

## 🛠 Technologies Used

- Java (JDK)  
- Object-Oriented Programming (Classes, Methods, Encapsulation)  
- ArrayList for Transaction Tracking  
- Scanner for User Input
